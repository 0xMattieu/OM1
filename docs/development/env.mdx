## Optional Environment Variables

- `ETH_ADDRESS`: The Ethereum address of agent, prefixed with `Ox`. Example: `0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045`. 
<Note>Only relevant if your agent has a wallet.</Note>
- `UNITREE_WIRED_ETHERNET`: Your network adapter that is connected to a Unitree robot. Example: `eno0`. 
<Note>Only relevant if your agent has a physical (robot) embodiment. You can set this to "SIM" to debug some limited functionality.</Note>
