## Speech and TTS

The Speech and TTS action plugin allows agents to speak aloud with a text-to-speech (TTS) system.

## Architecture

The Speech and TTS action plugin is composed of two main components as `move` action plugin:

- Interface Layer
- Connector Layer

```
[Speak Action Architecture]
├── Interface Layer
│   └── SpeakInput Dataclass
│       └── action: str
└── Connector Layer
    ├── ElevenLabs TTS
    │   ├── High-quality voice synthesis
    │   └── ASR integration
    ├── Riva TTS
    │   ├── NVIDIA's TTS engine
    │   └── ASR integration
    ├── Zenoh Publisher
    │   ├── Distributed messaging
    │   └── Topic-based publishing
    └── ROS2 Connector
        └── ROS2 message publishing
```

## Data Flow

```
    A[LLM Command] --> B[Speak Interface]
    B --> C[SpeakInput]
    C --> D[TTS Connector]
    D --> E[Audio Output]
```
